
The following steps detail the Gmapping algorithm.
Comments can be found in gmapping.cpp denoting functions associated with the steps.

1. Particles are given by pose X, Y, Theta
2. Move particles according to the motion model. Add noise.

3. "Nudge" particles, using a capped local search to maximize likelihood field score. (Either hit cap of number-of-iterations or max-distance-nudged.) Check every direction, move pose to highest score, repeat.
- (shelved) If it didn't move, shrink distance nudged.
- (shelved) Use Bilinear Interpolation to determine weighted distance between the 4 cells surrounding the hit point, to find a likelihood field score.
If scan matching fails, rely soley on motion model.

4. Sample the neighborhood (every combination of delta changes in x, y, theta) and find the covariance matrix (sum L(k) of each sample, which is equal to P(x|x-1, u) * P(z|x,m))

5. Calculate the new weight of each particle, which is its old weight * covariance matrix

(shelved) 6. If new weight is 0 or rounds to 0, make pose simply based on raw odometry, and give a low penalty weight (caused impossible situation, like in a wall)

7. Calculate the new pose as a an average center of gravity based on samples and L(k) (S(a) * L(ka) + ... / covariance)

8. Normalize weights of samples

9. If Neff is below a certain value (Neff threshold, like N/2) resample with replacement

10. Resample via "low variance resampling", like a comb on a number line rather than darts at the number line

11. After resample, reset all weights to 1/N to become a valid belief (note: we do NOT normalize particle weights if we don't resample)

12. Update maps according to current sensor scan (don't update the same cell twice within one scan; hits trump misses)

(13. Create render copy (graphics))

scoring function: e^-(d^2 / (2*sigma^2))

Gmapping must track a history of moves, to update particles after they've been "nudged" correctly.

As well, you must track the following:
- where was this particle when you asked me to update?
- point cloud with timestamp
- Where was each particle when the timestamp ended
- Where was each particle when the timestamp started

for(particle)

Turn range/theta into x, y
find min x,y and max x,y
add safety to max x,y; subtract safety from min x,y
pull all patches from min x,y to max x,y (plus the little on each end)
flatten into 1D array
pass to 2D transform
pull d from distance transform
Any d outside region is considered as if it hit the edge; either shorten ray or consider a miss.

run local search
scan neighbors and determine L(k)s, add to P(Z) and P(X) matricies
